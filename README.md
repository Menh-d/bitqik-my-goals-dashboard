# Bitqik — MY GOALS Strategic Dashboard

A premium, interactive strategic action dashboard built with **Neumorphism UI** design.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🗂️ **4 Tabs** | ແຜນຍຸດທະສາດ / Roadmap / Fishbone / Workshop Board |
| 🐟 **M-Framework** | 4Ms, 5Ms, 6Ms, Custom Fishbone Analysis |
| 🗒️ **Workshop Board** | Sticky Notes with Drag & Drop |
| 🌙 **Dark / Light Mode** | Neumorphism theme toggle |
| 📤 **Export PPTX** | Auto-generate 5-slide PowerPoint |
| 💾 **localStorage** | Data persists in browser |
| 📋 **Multi-Project** | Manage multiple projects |

## 🛠️ Tech Stack

- **Alpine.js** — Reactive UI
- **Tailwind CSS** — Utility styling
- **PptxGenJS** — PPTX export
- **Lucide Icons** — Icon library
- **Noto Sans Lao** — Lao font support

## 🚀 Run Locally

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```

## 🎨 Design

- **Style**: Neumorphism (Dark/Light)
- **Primary Color**: `#F47A20` (Bitqik Orange)
- **Framework**: 4Ms / 5Ms / 6Ms / Custom
